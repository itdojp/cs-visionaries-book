### 特に注意が必要な記述

#### 高リスク記述例
❌ **避けるべき記述**:
- 「関係者の証言によると...」（出典不明）
- 「社内では...と言われている」（未確認情報）
- 企業の内部事情の詳細な推測

✅ **安全な記述例**:
- 「〇〇氏は△△において次のように発言している」（出典明記）
- 「公開された資料によると...」（検証可能）
- 一般に知られている技術的事実

#### 引用のガイドライン
1. **短文引用**: 1回につき50文字以内
2. **出典明記**: 書籍名、著者、ページ数を必ず記載
3. **引用目的**: 批評・研究目的であることを明確化
4. **引用比率**: 本文に対する引用の割合を適切に保つ

### 法的安全性を高める執筆方針

#### 基本原則
1. **事実ベース**: 推測や憶測を避ける
2. **複数ソース**: 1つの事実に対し複数の情報源
3. **中立的視点**: 特定の企業・人物を過度に批判しない
4. **教育目的**: 学習・啓発が目的であることを明確化

#### 具体的な記述テクニック
```markdown
❌ 避けるべき表現:
「ジョブズは激怒して会議室を出て行った」

✅ 推奨表現:
「〇〇氏の自伝によると、ジョブズは会議を中断したとされている」
```
