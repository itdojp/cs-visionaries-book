# デジタル革命の舞台裏
## コンピュータサイエンスを築いた天才たちの物語

### スマートフォンの中に眠る天才たちの遺産

あなたが毎日手にするスマートフォン。その小さな画面の向こうには、200年にわたる人類の挑戦と革新の歴史が刻まれています。

朝のアラーム、タッチスクリーン、メッセージ、検索、写真撮影—これら全ての技術は、時代を超えた天才たちの夢と情熱の結晶です。

## 概要

本書では、デジタル時代を創造した10人（組）の革新者たちの人間ドラマを通じて、現代テクノロジーの本質とビジネスへの示唆を探ります。エイダ・ラブレスからスティーブ・ジョブズ、現代のAI研究者まで、彼らの挑戦、苦悩、そして偉大な功績から学ぶ一冊です。

### 取り上げる革新者たち

- **エイダ・ラブレス** (1815-1852) - 世界初のプログラマー
- **アラン・チューリング** (1912-1954) - 計算理論とAIの父
- **グレース・ホッパー** (1906-1992) - コンパイラの発明者
- **嶋正利・高橋秀俊** - 日本の技術革新を世界に示した技術者
- **スティーブ・ジョブズ&ウォズニアック** - パーソナルコンピュータ革命
- **ティム・バーナーズ＝リー** (1955-) - World Wide Webの発明者
- **ラリー・ペイジ&セルゲイ・ブリン** - Google創業者
- **マーク・ザッカーバーグ** (1984-) - Facebook創業者
- **ジェフ・ベゾス** (1964-) - Amazon創業者、クラウドの普及者
- **現代のAI研究者たち** - 人工知能に魂を宿そうとする錬金術師

## 本書の特徴

- **現代との接続** - 各人物の功績が現在のテクノロジーにどう生きているか
- **人間ドラマ** - 成功だけでなく、失敗、苦悩、人間関係も描写
- **ビジネス実用性** - 現代のビジネスパーソンが学べる教訓を提示
- **ビジュアル重視** - 図解、写真、インフォグラフィックで理解を促進
